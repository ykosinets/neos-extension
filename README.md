## Neos Fusion Tools

### About
VS Code extension providing advanced language support for Neos Fusion:
- Prototype go-to-definition
- Prototype autocomplete
- Prop go-to-definition
- Prop autocomplete

This is the initial scaffold.
<hr>

### Instalation
- A (via UI)
	1.	Open Cursor
	2.	Extensions panel
	3.	⋯ → Install from VSIX
	4.	Select neos-fusion-tools-{version}.vsix 
- B (via terminal/console)
    1.  Ctrl/Cmd + Shift + P → Shell Command: Install "code" command (in PATH)
    2.  ```cd {path-to}/neos-fusion-tools-{version}.vsix``` 
    3.  ```code --install-extension neos-fusion-tools-{version}.vsix```
